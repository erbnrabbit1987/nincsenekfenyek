# Frontend - Nincsenek Fények!

> React frontend alkalmazás

## Tech Stack (Tervezett)

- **React 18+**
- **TypeScript**
- **Vite** - Build tool
- **Material-UI** vagy **Ant Design** - UI komponensek
- **React Query** - Server state management
- **Axios** - HTTP client

## Projekt Struktúra (Tervezett)

```
frontend/
├── src/
│   ├── components/      # React komponensek
│   ├── pages/          # Oldalak
│   ├── services/       # API hívások
│   ├── hooks/          # Custom hooks
│   ├── utils/          # Segédfüggvények
│   ├── types/          # TypeScript típusok
│   └── App.tsx         # Fő komponens
├── public/             # Statikus fájlok
├── package.json
└── vite.config.ts
```

## Telepítés (Később)

A frontend projekt később lesz inicializálva:

```bash
cd frontend
npm create vite@latest . -- --template react-ts
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
